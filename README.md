# AtelierStefanKrauss — lokal entwickeln

To download sample photos into `public/assets/icons/` run:

```bash
chmod +x scripts/fetch-photos.sh
scripts/fetch-photos.sh
```

If you prefer your own images, place them as `public/assets/icons/photo1.jpg` … `photo5.jpg`.

## Uploads / dist

Lege alle Dateien, die auf den Server hochgeladen werden sollen, in den `dist/`-Ordner. Beispielsweise:

- `dist/.env.example` — Beispiel-Konfigurationsdatei
- `dist/` kann weitere Produktionsassets oder Deployment-Dateien enthalten

Vor dem Upload: Inhalte in `dist/` prüfen und sensible Daten entfernen oder ersetzen.
